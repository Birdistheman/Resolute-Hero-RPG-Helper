Resolute Hero RPG HTML Helper

This HTML file is a fan-made companion tool for the mobile game Resolute Hero RPG.
This tool can be used directly in-browser through GitHub Pages or downloaded and used offline locally.

The purpose of the file is to provide:

enemy lookup information,

equipment and accessory data,

EXP calculations,

progression planning,

and farming optimization tools.



---

Features

Current HTML functionality includes:

Enemy database

Enemy EXP values

BP cost listings

Equipment lookup

Accessory lookup

Rune information

Progression reference data

Farming comparison support



---

Main Purpose

The main goal of the HTML helper is to determine:

> Which enemy is the most efficient target for your current build?



This includes balancing:

EXP bonuses

BP costs

progression difficulty

and farming efficiency.



---

EXP Efficiency Formula

(Base Enemy EXP × Total EXP Multiplier) ÷ BP Cost

EXP Multiplier

1 + (Bonus EXP % ÷ 100)

Examples:

+800% EXP = 9x total EXP

+1400% EXP = 15x total EXP

+2420% EXP = 25.2x total EXP
